# Rotate-Click Carousel for Marketing Landing Page Layouts

**Issue:** #50467

## Description

A pure CSS Rotate-Click Carousel designed for marketing landing pages. Cards rotate and slightly scale during interaction without JavaScript.

## Features

- Pure HTML & CSS
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- Lightweight
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

```html
<div class="carousel">
    <div class="card">...</div>
</div>
```

## Customization

```css
:root{
    --rotate:10deg;
    --duration:.4s;
    --scale:1.08;
}
```
