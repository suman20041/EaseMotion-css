# CSS Scale-Hover Carousel for Creative Portfolio Layouts

A responsive, pure CSS carousel designed for showcasing creative portfolio content with smooth scale-hover interactions. The component emphasizes elegant motion while remaining lightweight, accessible, and easy to customize.

## Overview

The Scale-Hover Carousel demonstrates how CSS alone can be used to create engaging interactive layouts without relying on JavaScript. Cards gently scale and gain visual emphasis on hover or keyboard focus, helping users naturally discover featured content.

The component is built using modern CSS techniques including Grid, Scroll Snap, transitions, and CSS Custom Properties.

---

## Problem Statement

Modern portfolio websites often require interactive project galleries that feel engaging while maintaining excellent performance.

Many carousel implementations depend on JavaScript libraries, increasing bundle size and implementation complexity for interactions that can be achieved using CSS alone.

This example demonstrates a lightweight alternative that provides polished animations with zero JavaScript.

---

## Features

- Pure CSS implementation
- Smooth scale-hover interaction
- Keyboard-accessible focus states
- Responsive layout for desktop and mobile
- Horizontal scrolling with CSS Scroll Snap
- Customizable using CSS Custom Properties
- Supports `prefers-reduced-motion`
- No external libraries or frameworks required

---

## Customization

Animation and layout behavior can be adjusted using CSS custom properties.

Examples include:

- Card width
- Carousel spacing
- Hover scale factor
- Hover lift distance
- Transition duration
- Transition easing

This allows the component to integrate easily with different design systems while keeping the implementation simple.
