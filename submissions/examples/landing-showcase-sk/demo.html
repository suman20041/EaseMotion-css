Features

Everything you need to animate

Purpose-built utilities that make animation composable, readable, and production-ready.

🎬

30+ Animations

Fade, slide, zoom, bounce, spin, pulse, shake, wobble, flip, float, heartbeat, and more — all with a single class.

⚡

Zero Dependencies

Pure CSS. No JavaScript required. No build step. Just drop one file and start animating everything.

🎨

Human-Readable

Classes like ease-fade-in, ease-slide-up, and ease-hover-lift read like plain English.

📱

Responsive Utilities

Mobile-first breakpoints at sm, md, lg, and xl. Flexbox and grid utilities that adapt to any screen.

🌓

Dark Mode Ready

Automatic prefers-color-scheme detection plus data-theme manual control built right in.

♿

Accessible

Respects prefers-reduced-motion. Focus rings, ARIA-friendly components, and screen-reader utilities.
