# Ease Wobble Card Hover

## Overview

A reusable wobble hover animation component built using pure HTML and CSS.

The component demonstrates a modern wobble animation suitable for cards in portfolios, SaaS products, landing pages, dashboards, blogs, and e-commerce websites.

---

## Features

- Smooth wobble hover animation
- Glassmorphism-inspired card design
- Responsive layout
- Pure HTML & CSS
- Modern UI
- Easy customization

---

## Browser Support

- Chrome
- Edge
- Firefox
- Safari
