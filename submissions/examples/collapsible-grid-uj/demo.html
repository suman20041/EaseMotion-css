Accordion Smooth Slide-Down

Click on the items below. The details expand and collapse smoothly using CSS grid rows, without fixed heights or JavaScript.

How does the pure CSS transition work? ▼
By utilizing a two-row CSS grid containing `grid-template-rows: auto 0fr` transitioning to `auto 1fr` on open, the browser automatically interpolates the height of the content block dynamically without layout jumps.
Why is this better than max-height? ▼
Traditional `max-height` transitions require developers to declare large arbitrary values (e.g., `max-height: 1000px`), causing noticeable animation delays and transition speed inconsistencies depending on content size.
