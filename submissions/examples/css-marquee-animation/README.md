# CSS Marquee Animation Example

## What does this do?
This example creates a smooth, continuous scrolling text animation using pure CSS.  
It replicates the classic `<marquee>` behavior in a modern way, allowing text to move horizontally across the screen with optional pause on hover.

## How is it used?
You simply add the `marquee` class to a container and place your text inside.  
The CSS animation automatically handles the scrolling effect, so no JavaScript or external libraries are required.  
You can also extend it with modifiers (like faster speed or reverse direction) to suit different design needs.

## Why is it useful?
This feature replaces the outdated `<marquee>` tag with a clean, beginner‑friendly CSS solution.  
It fits perfectly with EaseMotion CSS’s philosophy of **human‑readable, animation‑first utilities** by offering a simple way to add continuous motion effects.  
Scrolling text is useful for banners, announcements, news tickers, or any UI element that needs to grab attention.  
By keeping the code lightweight and easy to understand, it helps newcomers learn CSS animations while expanding the library with a versatile effect.
