# Micro-Gravity Orbital Manufacturing Terminal

A futuristic responsive HTML & CSS showcase demonstrating an orbital manufacturing control dashboard designed for the EaseMotion CSS project.

## Features

- Responsive layout
- Animated orbital rings
- Glassmorphism interface
- Production analytics dashboard
- Manufacturing module cards
- AI robotics control center
- Satellite network visualization
- Live manufacturing activity timeline
- Modern hover effects
- Pure HTML5 & CSS3

## Technologies

- HTML5
- CSS3
