# Skew-Active Carousel – Responsive Dashboard

A pure CSS Skew-Active Carousel component built for EaseMotion CSS.

## Features

- Pure CSS (No JavaScript)
- Responsive dashboard-inspired layout
- Smooth Skew-Active transition
- Keyboard accessible
- Uses CSS custom properties
- Easy to customize

## Custom Properties

```css
--ease-duration
--ease-easing
--ease-skew
--ease-scale
--primary
--secondary
```
