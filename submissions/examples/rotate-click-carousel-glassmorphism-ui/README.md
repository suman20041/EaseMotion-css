# CSS Rotate-Click Carousel – Glassmorphism UI

A modern Glassmorphism Rotate-Click Carousel built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Glassmorphism design
- Rotate-click animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

Open `demo.html` directly in any browser.

Customize using CSS variables:

```css
:root{
  --rotate:8deg;
  --duration:.35s;
}
```
