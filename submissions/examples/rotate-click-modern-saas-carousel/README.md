# Rotate-Click Carousel for Modern SaaS Layouts

**Issue:** #50465

## Description

A responsive pure CSS Rotate-Click Carousel created for Modern SaaS layouts. Cards smoothly rotate and scale during interaction using only HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Lightweight
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

```html
<div class="carousel">
    <div class="card">...</div>
</div>
```
