Ease Tabs — Fixed (No Memory Leak)

This demo fixes the bug where initializeTabs() re-attached a window resize listener and per-input change listeners on every MutationObserver trigger, causing listeners to accumulate indefinitely.

Overview panel content. Resize the window — the underline recalculates smoothly, with only one resize listener bound.

Details panel content. Try clicking "Mutate DOM" below repeatedly, then check DevTools console for the listener count.

Settings panel content. No matter how many mutations occur, the resize listener count stays at 1.
