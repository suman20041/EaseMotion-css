# Progress Ring Animated

Animated circular progress ring with SVG stroke-dashoffset. Percentage displayed in center with gradient stroke.

## Features

- SVG circle progress ring
- Stroke-dashoffset animation controlled by JS
- Percentage display in center
- Gradient stroke color

## Usage

Set `--pra-pct` (0-100) and adjust `stroke-dashoffset` on `.pra-fill` circle. JS handles slider input.
