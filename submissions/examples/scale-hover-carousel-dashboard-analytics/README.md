# CSS Scale-Hover Carousel – Dashboard Analytics

A responsive **Dashboard Analytics Carousel** built with pure HTML and CSS featuring smooth scale-hover animations for analytics cards.

## Features

- Pure HTML & CSS
- Dashboard analytics card layout
- Smooth scale-hover animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#2563eb;
  --scale:1.08;
  --duration:.35s;
}
```

## Folder Structure

```
scale-hover-carousel-dashboard-analytics/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser and customize colors, animation speed, scale, and spacing using CSS variables.
