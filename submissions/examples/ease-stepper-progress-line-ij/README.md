# Stepper Progress Line

Multi-step progress indicator with numbered circles and connecting line. Line fills as user progresses via --spl-step.

## Features

- Numbered step circles
- Connecting line between steps
- Line fills as steps complete
- Smooth step transitions via --spl-step

## Usage

Set `--spl-step` (0-3) on `.spl-stepper`. CSS colors active steps and fills connecting line.
