CSS-only dropdown menu with slide+fade animation — Issue #14162

Alignment Variants
Form Select Style

How it works: The dropdown uses :focus-within — when the trigger button inside .ease-dropdown receives focus (click or Tab), the menu becomes visible via opacity: 0→1 and translateY(-6px)→0. Clicking outside or pressing Escape (on the trigger) closes it. No JavaScript required.

Items are <button> elements for keyboard accessibility. Arrow key navigation works natively. The divider uses .ease-dropdown-divider.
